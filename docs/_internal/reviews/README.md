# Archived Internal Reviews

## Purpose

This directory keeps selected, closed planning reviews whose decision history
is still useful but whose contents must not be treated as active proposals or
current runtime contracts.

Only explicitly allowlisted archive files are versioned. Ad-hoc review notes
remain local-only. Active design work belongs in `../proposals/`; current
ownership and behavior belong in `../project-map/`; reusable QA knowledge
belongs in `../test-map/`.
